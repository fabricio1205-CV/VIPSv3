# Casa Victor | Seguimiento VIP

## Qué incluye
- Login por usuario + PIN
- Perfil Administrador y perfil Vendedor
- Lectura automática de `vips.csv`
- Soporte para CSV con separador `;`, `,` o tabulación
- Mes en texto
- Resaltado en rojo cuando la venta es 0 o menor
- Estado obligatorio para no compra
- Comentarios opcionales
- Guardado local automático
- Botón **Enviar Informe**
- Filtros por mes, año, vendedor, estado, enviados/no enviados

## Archivos que tenés que subir al repo
- `index.html`
- `styles.css`
- `app.js`
- `config.js`
- `vips.csv`
- `casavictor-logo.png`

## Cómo publicarlo
1. Subí todo a la raíz del repo
2. En GitHub: **Settings > Pages**
3. Fuente: `Deploy from a branch`
4. Branch: `main`
5. Folder: `/ (root)`
6. Guardá
7. Esperá 20 a 40 segundos
8. Abrí la URL de GitHub Pages
9. Hacé `Ctrl + F5`

## Carga semanal
Reemplazá el archivo `vips.csv` por el nuevo y luego apretá **Actualizar base desde CSV**.

## Más adelante
Cuando quieras, se reemplaza el guardado local por Google Sheets + Apps Script.
